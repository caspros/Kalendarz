# Kalendarz
Prosty projekt kalendarza na Inżynierię Oprogramowania - Kacper Śledzik i Aleksander Szuba
Projekt wykonany w php

## Features 
- Export to .ics file
- Add weekly events
- Delete events

## Installation
Create database calendar in phpmyadmin
Set up database configuration in connect.php
